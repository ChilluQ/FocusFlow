# 🚀 FocusFlow

**FocusFlow** is a modern student productivity web application designed to help students stay organized, focused, and consistent with their academic goals.

It brings task management, study planning, focus tracking, and productivity streaks together in one clean and user-friendly dashboard.

## 🚀 Getting Started

Follow these steps to run FocusFlow locally.

### 1. Clone the repository

```bash
git clone https://github.com/ChilluQ/FocusFlow.git
```

### 2. Navigate to the project directory

```bash
cd FocusFlow
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm run dev
```

Open the local development URL displayed in your terminal.

## 🎯 Project Goal

The goal of FocusFlow is to create an all-in-one productivity platform specifically designed for students.

Instead of switching between multiple applications for tasks, assignments, study planning, notes, and focus sessions, FocusFlow aims to bring essential student productivity tools together in one unified platform.
